# rondiver.com

Personal site for Ron Diver. Landing page with potential blog expansion.

## Stack
- **Framework:** Astro 5
- **Hosting:** Vercel (via `@astrojs/vercel`)
- **Style:** Minimal, clean. No CSS framework — scoped styles in components.

## Structure
- `src/layouts/Base.astro` — Base HTML layout with global styles
- `src/pages/index.astro` — Landing page

## Commands
- `npm run dev` — Dev server
- `npm run build` — Production build
- `npm run preview` — Preview production build

## Domain
- rondiver.com (configure in Vercel dashboard after first deploy)
